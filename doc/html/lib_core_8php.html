render ( $  view,
$  vars = array() 
)

Rend la vue demandée en y affectant les différentes variables définies dans le contrôleur et génération des liens vers les fichiers .css et .js. A utiliser depuis le contrôleur.

Auteur:
Pierre Criulanscy
Depuis:
0.1.1
Paramètres:
$viewnom de la vue à rendre
$varsvariables à faire passer à la vue sous la forme d'un tableau associatif. La clé représente le nom de la variable qui pourra être utilisée dans la vue, sa valeur correspond à la valeur de la clé dans le tableau. Si $vars = array("var1" => "toto", "var2" => "titi"); alors dans la vue seront accessibles les variables $var1 et $var2 avec comme valeur respective "toto" et "titi"
